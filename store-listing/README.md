# Store Listing — Google Play (and App Store)

Copy + asset spec for the public store listing. Everything here is plain
text or markdown so legal / marketing review can edit without touching the
codebase.

## Files

- `play-store.md` — Google Play Console copy (short + full description,
  what's-new template, keywords, content rating notes).
- `app-store.md` — App Store Connect copy (subtitle, promotional text,
  description, keywords).
- `assets-spec.md` — exact pixel sizes, formats, naming conventions, and
  capture instructions for icon / feature graphic / screenshots.
